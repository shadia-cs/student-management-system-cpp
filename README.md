\# 📚 Simple C++ Student Management System



A Simple C++ program to add, view, and delete students.  

Perfect for learning C++ basics like structs, vectors, enums, and console I/O.







\## ⚡ Features



\- Add a student (name, ID, GPA)  

\- View all students  

\- Delete a student by ID  

\- Easy console menu





\## ▶️ How to Run

```bash

git clone <your-repo-link>

```



```bash

cd student-management-system

```

```bash

g++ main.cpp -o student\_system

```



```bash

./student\_system      # Linux / Mac

student\_system.exe    # Windows

```
